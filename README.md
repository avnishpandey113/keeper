## Installation

Clone it!

```
$ git clone https://github.com/avnishpandey113/keeper.git
```

Install project dependencies:

```
$ npm i
```

Go into the project directory using git-scm/hyperjs and run the command:

```
$ npm start
```

Open `http://localhost:3000` and enjoy!
